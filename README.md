# Scrapyard Tycoon (ROBLOX)

Scrapyard Tycoon is a small Roblox tycoon prototype about building up an old scrapyard one unlock at a time.

## Gameplay Loop

Unlock the scrapyard, buy Broken Cars, collect StoredParts at the CollectPad, and spend Parts to expand.

## Current Features

* Touch-based build buttons
* Broken Car production
* Shared StoredParts collection
* CollectPad Parts transfer
* World-space SurfaceGui signs
* Whole-number Parts display
* Workbench income boost
* Scrapyard expansion stages

## Development Notes

Roblox Studio is the source of truth for layout, objects, tags, visuals, BuildCost, and DisplayName values.
